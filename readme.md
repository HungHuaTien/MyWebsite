# 🧠 網站資源總覽

歡迎來到本資源庫！這裡是我的 AI 部落格。  
所有內容依照用途與功能分類為不同子資料夾，方便未來快速定位、版本控制與共享。

> ✨ **原則：Less is more, every file has a home.**  
> 無論是上課講義、學習筆記、心智圖還是文章，都能找到對應的分類。

---

## 📁 資料夾結構與說明

### 📦 `/handouts/` — 可供輸出下載的檔案

- ✅ 發給學生的講義、作業等PDF檔案
- ✅ 多為定稿、已排版或轉換格式版本

### 📘 `/docs/` — 自己的學習筆記

- ✅ 整理過的 `.qmd`檔案
- ✅ 含 YAML、段落設計、Quarto 設定與標記語法

### 📝 `/notes/` — 補充說明資料

- ✅ 個人反思、設定備忘與隨筆
- ✅ 通常以 `.md` 格式為主，含標籤與 YAML metadata

### 🖼 `/figures/` — 所有圖片集中管理

- ✅ 插圖、流程圖、Quarto 輸出引用圖
- ✅ 支援引用於講義、教材或 GitHub Pages 網頁


---

## 🔧 使用建議

- 所有檔案建議加上 YAML metadata，便於後續分類與查找
- 每個資料夾皆可搭配 `README.md` 作為說明與索引
- 若有需要，可延伸整理為 Quarto 網頁或 GitHub Pages 教案網站
